# rgstr
NFC based registration system for schools

Year 13 A level project. Signs students in or out of a short leave absence database by scanning NFC cards.

## System requirements:

RGSTRControlpanel.py
-Python 3.4+
-mysql-connector for python3.4+
-A MySQL server.

RGSTR.py
-Python 3.4+
-mysql-connector for python3.4+
-nxppy
-Raspbian Jessie+
-gTTS
-A MySQL server.
